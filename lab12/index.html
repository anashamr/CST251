Float

Float property allows an element to reposition right or left and have text flow around it. It's a bit tricky when there is more than one floated element. In order to do this you must use float: right; or float: left;.

Center Horizontal

Centering in HTML is a bit difficult and requires some tricks. However, centering text horizontally is pretty simple to do. All that has to be done is text-align: center;. Centering elements horizontally whta must be done is display: block; , margin-left: auto; , then margin-right: auto;.
